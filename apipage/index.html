⚙️ Codex 安装教程

选择你的操作系统，按顺序复制执行以下命令即可完成安装配置。

已在 Ubuntu 20.04 测试通过

# 1. 安装 Node.js 与 npm
sudo apt install nodejs npm -y

# 2. 配置 NodeSource 22.x 源
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -

# 3. 安装 Node.js
sudo apt install -y nodejs

# 4. 全局安装 Codex
sudo npm install -g @openai/codex

# 5. 运行一键配置脚本
bash <(curl -fsSL https://cdn.jsdelivr.net/gh/luohongk/luohongkun-api-setup@main/codex.sh)

在 PowerShell 中执行以下命令

# 1. 安装 Node.js 22.x（使用 winget）
winget install OpenJS.NodeJS.LTS

# 若无 winget，可前往 https://nodejs.org 下载安装包手动安装

# 2. 全局安装 Codex
npm install -g @openai/codex

# 3. 运行一键配置脚本
powershell -ExecutionPolicy Bypass -Command '$r = Invoke-WebRequest -UseBasicParsing "https://cdn.jsdelivr.net/gh/luohongk/luohongkun-api-setup@main/codex.ps1"; $c = $r.Content; if ($c -is [byte[]]) { $c = [System.Text.Encoding]::UTF8.GetString($c) }; & ([scriptblock]::Create([string]$c))'

🔄 API Key 更新

当你在控制台重置或更换了 API Key 后，按下方命令更新本地环境变量即可生效。

将下面命令中的密钥替换为你的新 API Key（bash 用户请把 ~/.zshrc 换成 ~/.bashrc）

# 1. 写入新的 API Key 到环境变量
echo 'export LUOHONGKUN_API_KEY="你的新APIKey"' >> ~/.zshrc

# 2. 使配置立即生效
source ~/.zshrc

在 PowerShell 中执行以下命令

# 1. 写入新的 API Key 到用户环境变量（永久生效）
[System.Environment]::SetEnvironmentVariable("LUOHONGKUN_API_KEY", "你的新APIKey", "User")

# 2. 使当前会话立即生效
$env:LUOHONGKUN_API_KEY = "你的新APIKey"
